Frequently Asked Questions

Q: Which file formats are supported?
A: PDF, JPG, PNG, HEIC, WebP, and compressed PDF/Images.
Q: Is my data safe?
A: Yes. All conversions happen on your device. No files are uploaded to any server.
Q: Why is the app asking for photo library access?
A: This permission is needed to select images for conversion. You can choose "Selected Photos" to limit access.
Q: The converted file is not what I expected.
A: Please make sure the source file is not password-protected or corrupted. Contact us if the issue persists.
Q: How do I restore my purchase?
A: Go to Settings → Restore Purchase. Make sure you are signed in with the same Apple ID used for the original purchase.
